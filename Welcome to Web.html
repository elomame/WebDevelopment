WELCOME TO Web Development

Elom Amedome project for the school of GordonState the Web Dev HTML Project. This project is designed to help you understand the basics of HTML coding. With that in mind with the job of A front-end web developer is responsible for implementing visual elements that users see and interact with in a web application. They are usually supported by back-end web developers, who are responsible for server-side application logic and integration of the work front-end developers do.

Given below is a list of things you need to do to become a Awesome front-end web developer:

  • A title in the title bar
  • The body of the document should have a background. This may be a color, a public domain image or an image for which you own the copyright.
  • Use heading text at the top of the document body (use the <H#> tag)
  • Add at least four sentences of text
  • Include a list with at least 3 items (like this one!). To create the list, you must use the list tags.
  • Add a short summary of your powerpoint presentation and make a link to the PDF version of it.
  • Make four links to other webpages:
    1. One link to the W2 School homepage (https://www.w3schools.com/html/html_form_input_types.asp)
    2. One link to the persuasion page you will make later in the project.
    3. Two different links to sites that contain information you are interested in.
  • Add two pictures to the page.
